
# Trợ Lý Prompt Infographic Giáo Dục (Offline Mode)

Ứng dụng hỗ trợ giáo viên tạo prompt chất lượng cao để thiết kế Infographic bằng AI (Midjourney, DALL-E, v.v.) thông qua các mẫu điền sẵn chuẩn hóa.

## Tính năng
- **Offline Mode**: Điền biểu mẫu để tạo prompt theo khuôn mẫu chuẩn.
- **Tùy chỉnh đa dạng**: Hỗ trợ nhiều phong cách (Đồng quê VN, Cyberpunk, Flat Design...), cảm xúc và mục tiêu giáo dục.
- **Lịch sử Prompt**: Tự động lưu 10 prompt gần nhất vào trình duyệt.
- **Không cần Internet**: Hoạt động hoàn toàn trên trình duyệt người dùng, không cần API Key hay kết nối server.

## Cài đặt và Chạy (Local)

1. Clone dự án.
2. Cài đặt thư viện:
   ```bash
   npm install
   ```
3. Chạy server phát triển:
   ```bash
   npm run dev
   ```

## Deploy lên Vercel

1. Đẩy code lên GitHub/GitLab/Bitbucket.
2. Vào dashboard Vercel -> **Add New Project**.
3. Import repository vừa đẩy.
4. **Framework Preset**: Chọn **Vite**.
5. Nhấn **Deploy**.
